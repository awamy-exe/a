# 🐾 PetBook

Sistema para cadastrar e gerenciar seus pets com fotos, informações e muito amor!

## ✨ Funcionalidades

- 📷 Upload de fotos dos pets
- 🎂 Data de nascimento e cálculo automático de idade
- 🎨 Cor, peso, tamanho e raça
- 🔍 Busca por nome, raça ou cor
- 🏷️ Filtros por espécie (cães, gatos, aves, roedores, etc)
- 📝 Observações e notas
- 💾 Dados salvos no navegador (localStorage)
- 📱 Design responsivo

## 🚀 Como usar

Abra o arquivo `index.html` no navegador!

## 🛠️ Tecnologias

- HTML5
- CSS3 (Glassmorphism + Dark Mode)
- JavaScript Vanilla
- localStorage para persistência
